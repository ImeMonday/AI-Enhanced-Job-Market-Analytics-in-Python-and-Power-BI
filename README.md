
AI Enhanced Job Market Analytics in Python and Power BI

Project Overview
This project analyzes global tech job listings using Python and Power BI.
You extract insights on pay trends, remote roles, required skills, and salary expectations.
The workflow builds a full data pipeline from raw CSV to a finished business dashboard.

Key Objectives
Clean and prepare raw job listing data
Engineer features for skills, seniority, remote status, and pay
Build salary prediction indicators
Create KPI measures in Power BI for remote pay, top skills, and job demand
Design an interactive dashboard for hiring managers and job seekers
